O programa foi implementado em python3 utiliza as bibliotecas sympy e numpy.
Para instalar as dependências, basta executar o script install_requirements.py por meio da linha de comando (executar na raiz do diretório):
    <path_to_python3> -u install_requirements.py

Após instalar as dependências, pode-se executar o programa pelo comando:
    <path_to_python3> -u main.py <input_file>

Obs.: Deve-se substituir as expressões da forma <expressão> pelo respectivo valor desejado.
